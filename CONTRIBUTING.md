# Contributing to KeetaNet

Спасибо за интерес к KeetaNet!

## Как внести вклад
1. Форкните репозиторий
2. Создайте ветку feature/*
3. Сделайте изменения
4. Создайте Pull Request

## Требования
- Чистый код
- Документация к каждому изменению
- Тесты (если применимо)
- Соблюдение KMSL‑1.0

## Запрещено
- Вносить изменения в лицензию
- Удалять авторские атрибуции
- Добавлять проприетарный код без разрешения
